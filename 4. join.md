### Join 

```sql
SELECT
   goal.player,
   goal.teamid,
   game.stadium,
   game.mdate 
FROM
   (
      game 
      JOIN
         goal
   )
   ON ( game.id = goal.matchid) 
WHERE
   goal.teamid = 'GER'
```

### Join with group by

```sql
SELECT teamname,
       Count(goal.player)
FROM   eteam
       JOIN goal
         ON eteam.id = goal.teamid
GROUP  BY eteam.teamname; 
```
